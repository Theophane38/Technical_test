# Test technique de Théophane Duval

## Description

Test technique rélaisé par Théophane Duval pour le poste de développeur front-end chez Eldo


## Commands

### before starting

```
    npm install
```

### start project

```
    npm run start
```

### build staging

```
    npm run build_staging
```

### build production

```
    npm run build_prod
```
